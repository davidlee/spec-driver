- **Delta**: DE-026 - Use zigmarkdoc for Zig documentation generation
- **Specs Impacted**: SPEC-124 (Sync command / contract generation)

- **Desired Outcome**: ZigAdapter generates contracts using zigmarkdoc subprocess calls instead of manual parsing

## 2. Context & Constraints

- **Current Behaviour**: ZigAdapter manually parses Zig files with regex, missing complex patterns
- **Target Behaviour**: Call zigmarkdoc tool (following GoAdapter pattern with gomarkdoc)
- **Dependencies**: zigmarkdoc installed at ~/.local/bin/zigmarkdoc
- **Constraints**: Must maintain backward compatibility with contract locations and frontmatter
