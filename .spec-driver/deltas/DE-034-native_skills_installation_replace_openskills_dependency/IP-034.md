## 1. Summary

- **Delta**: DE-034 – Native skills installation: replace openskills dependency
- **Desired Outcome**: `skills sync` installs skills from the spec-driver package to configurable agent target dirs, prunes de-listed skills, and renders AGENTS.md — all without openskills.

## 2. Context & Constraints

- **Current Behaviour**: openskills npm tool installs to `.agent/skills/`; `skills sync` reads from there to generate AGENTS.md
- **Target Behaviour**: Skills read from package source (`supekku/skills/`), installed to `.claude/skills/` and `.agents/skills/` per config
- **Dependencies**: None
- **Constraints**: Must not touch user-created skills in target dirs

## 5. Implementation Plan

### Step 1: `paths.py` + `config.py` (independent)

**`supekku/scripts/lib/core/paths.py`** — add `get_package_skills_dir()`:

```python
def get_package_skills_dir() -> Path:
  import supekku
  return Path(supekku.__file__).parent / "skills"
```

**`supekku/scripts/lib/core/config.py`** — add to `DEFAULT_CONFIG`:

```python
"skills": {
  "targets": ["claude", "codex"],
},
```

### Step 2: `sync.py` — core changes (TDD)

**Replace** `SKILLS_CACHE_DIR` with target mapping:

```python
SKILL_TARGET_DIRS: dict[str, Path] = {
  "claude": Path(".claude") / "skills",
  "codex": Path(".agents") / "skills",
}
```

**New functions:**

- `get_package_skill_names(skills_source_dir) -> set[str]` — list valid skills in package
- `install_skills_to_target(skills_source_dir, target_dir, allowed_names) -> dict` — copy allowlisted skills, idempotent (compares SKILL.md bytes)
- `prune_skills_from_target(target_dir, package_skill_names, allowed_names) -> list[str]` — remove de-listed package skills; user skills safe
- `_skill_dir_matches(src, dest) -> bool` — compare SKILL.md content

**Refactor:**

- `_collect_skills(repo_root, skills_source_dir)` — accept source dir param
- `sync_skills(repo_root, *, skills_source_dir=None)` — orchestrates install + prune + AGENTS.md; optional param for testability

### Step 3: CLI + installer

**`supekku/cli/skills.py`** — report per-target install/prune counts.

**`supekku/scripts/install.py`** — call `sync_skills(target_root)` at end of `initialize_workspace()`.

### Step 4: Tests

Update all existing fixtures (replace `.agent/skills/` with `skills_source_dir=`). New tests for:

- `get_package_skill_names` (valid, empty, ignores non-skill dirs)
- `install_skills_to_target` (copies, idempotent, updates changed, creates dir)
- `prune_skills_from_target` (removes de-listed, ignores user skills, noop)
- `_skill_dir_matches` (identical, different, missing)
- `sync_skills` integration (targets, pruning, config-driven, unknown target warns)
- CLI output format
- Installer integration

### Step 5: Lint + full test suite

`just test && just lint && just pylint`
