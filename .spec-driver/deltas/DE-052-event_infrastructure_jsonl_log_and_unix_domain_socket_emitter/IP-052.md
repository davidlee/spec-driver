## 1. Summary

- **Delta**: DE-052 — Event infrastructure: JSONL log and Unix domain socket emitter
- **Design Revision**: DR-052 (approved)
- **Desired Outcome**: Every leaf-command CLI invocation emits a structured
  JSONL event to `.spec-driver/run/events.jsonl` and fire-and-forgets a
  datagram to `.spec-driver/run/tui.sock`. Agent sessions are attributable.
  Help/completion paths are excluded.

## 2. Context & Constraints

- **Current Behaviour**: No structured CLI trace exists
- **Target Behaviour**: Per DR-052 — process-boundary wrapper gated by
  `_command_invoked` flag, set via `Command.invoke` interception
- **Dependencies**: None
- **Constraints**: Stdlib only, zero-cost when no TUI, fail-silent
