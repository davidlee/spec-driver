## 1. Summary

- **Delta**: DE-054 — TUI track mode: real-time agent session following
- **Design Revision**: DR-054
- **Desired Outcome**: `t` toggles a Track view in the TUI showing a live
  event stream from CLI commands, filterable by agent session, with
  click-to-navigate to touched artifacts in the browser.

## 2. Context & Constraints

- **Current Behaviour**: DE-052 emits events to JSONL log + socket. DE-053
  provides browsable TUI. Nothing listens on the socket.
- **Target Behaviour**: TrackScreen receives live events, displays them with
  session colouring, supports filtering and cross-screen navigation.
- **Dependencies**: DE-052 (completed), DE-053 (completed)
- **Constraints**: No new package deps. TUI is read-only. Must not scuttle
  existing TUI instances.
