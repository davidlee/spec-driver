## 1. Summary

- **Delta**: DE-058 - govern pylint signal and document lint standard
- **Specs Impacted**: PROD-003
- **Desired Outcome**: Publish a default lint-governance standard and restore
  pylint signal by fixing broken recursive test-file ignores and adding compact
  report workflows.

## 2. Context & Constraints

- **Current Behaviour**: Nested tests still emit low-signal docstring and
  protected-access warnings because per-file ignore globs are not recursive.
- **Target Behaviour**: Test-only noise is suppressed where intended; repo-level
  pylint remains authoritative, lint posture is documented in STD-002, and
  compact summary commands reduce rework during implementation.
- **Dependencies**: Existing pylint tooling and standard registry support.
- **Constraints**: No broad repo-wide warning cleanup in this delta.
