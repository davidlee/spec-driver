## 1. Summary
- **Delta**: DE-064 — spec-driver doctor: workspace health diagnostics
- **Design Revision**: DR-064
- **Problems / Issues**: ISSUE-022, ISSUE-015
- **Desired Outcome**: `spec-driver workspace doctor` runs 6 check categories
  and produces actionable diagnostic output with appropriate exit codes

## 2. Context & Constraints
- **Current Behaviour**: No unified health check. `workspace validate` covers
  cross-references only. Dependency/config/structure problems discovered ad hoc.
- **Target Behaviour**: Single command with structured output across deps,
  config, structure, registries, refs, lifecycle categories.
- **Dependencies**: None (all infrastructure exists)
- **Constraints**: Delegate to `WorkspaceValidator` for refs (POL-001). No
  magic numbers (POL-002). Thin CLI pattern.
