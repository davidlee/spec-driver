## 1. Summary

- **Delta**: DE-066 — TUI drift ledger integration
- **Desired Outcome**: Drift ledgers appear in the TUI browser as a selectable
  artifact type with list and preview rendering.

## 2. Context & Constraints

- **Current Behaviour**: TUI has no knowledge of drift ledgers. `ArtifactType`
  enum lacks a drift entry. Registry, path mapping, and metadata are unwired.
- **Target Behaviour**: Drift ledgers selectable in TUI type selector, list
  shows id/name/status/entries/delta, preview renders detail view, file watcher
  refreshes on `.spec-driver/drift/` changes.
