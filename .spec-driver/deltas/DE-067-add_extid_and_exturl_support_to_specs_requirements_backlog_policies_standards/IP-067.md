## 1. Summary

- **Delta**: DE-067 – Add ext_id and ext_url support to artifact frontmatter
- **Specs Impacted**: PROD-004, SPEC-116, SPEC-122
- **Issues**: ISSUE-024
- **Desired Outcome**: All artifact types accept optional `ext_id`/`ext_url`
  frontmatter fields, visible via `list --external`, `show`, and JSON output

## 2. Context & Constraints

- **Current Behaviour**: No external reference fields exist on any artifact model
- **Target Behaviour**: Optional `ext_id` (string) and `ext_url` (string) on all
  artifact types; `list --external` shows ext_id column; `show` and JSON include
  both fields when present
- **Dependencies**: None
- **Constraints**: Purely additive. No migration. Fields optional with no default.
