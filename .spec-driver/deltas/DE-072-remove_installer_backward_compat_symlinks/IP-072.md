## 1. Summary

- **Delta**: DE-072 - remove installer backward-compat symlinks
- **Specs Impacted**: SPEC-129
- **Desired Outcome**: Installer behavior and tests align on canonical-only workspace creation.
- **Documentation Reconciliation**: Update ADR-006 so the recorded workspace-install behavior matches the code change in this delta.

## 2. Context & Constraints

- **Current Behaviour**: Fresh installs create compatibility paths such as `specify/`, `change/`, `backlog/`, and `memory/` in addition to canonical directories under `.spec-driver/`.
- **Target Behaviour**: Fresh installs stop creating those compatibility paths while retaining canonical workspace initialization.
- **Dependencies**: Installer code and installer tests only.
- **Constraints**: Keep the delta narrow and preserve existing canonical workspace creation.
