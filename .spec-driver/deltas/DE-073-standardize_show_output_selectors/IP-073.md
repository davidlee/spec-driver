## 1. Summary

- **Delta**: DE-073 — CLI output selectors, view UX, and resolve-links improvements
- **Design Revision**: [DR-073](./DR-073.md)
- **Problems / Issues**: ISSUE-035 (remaining), ISSUE-036
- **Desired Outcome**: Unified `--content-type` on `show`, rendered non-paged
  `view` by default with opt-in `--pager`, `read` alias, and scoped
  `resolve links` diagnostics.

## 2. Context & Constraints

- **Current Behaviour**: `show` has ad-hoc output flags; `view` always pages;
  `resolve links` is whole-corpus only.
- **Target Behaviour**: Per DR-073 design decisions DEC-073-01 through DEC-073-04.
- **Dependencies**: None (DE-045 is independent).
- **Constraints**: No breaking changes to existing flags. Reduce `view.py`
  duplication rather than compound it.
