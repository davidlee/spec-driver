## 1. Summary

- **Delta**: DE-087 — Cross-artifact TUI fuzzy search with relational weighting
- **Design Revision**: [DR-087](./DR-087.md)
- **Specs Impacted**: PROD-010 (CLI UX), PROD-015 (spec taxonomy and navigation)
- **Backlog**: IMPR-011 (TUI polish, navigation, and relational display)
- **Desired Outcome**: A single `/` keybinding opens a modal overlay that fuzzy-searches across all artifact types, including YAML frontmatter attributes and relation targets, with weighted scoring.

## 2. Context & Constraints

- **Current Behaviour**: TUI fuzzy search is per-type only (id/title/status). No cross-type or attribute search.
- **Target Behaviour**: Cross-type modal overlay with tiered scoring (DEC-087-01 through DEC-087-05).
- **Dependencies**: DE-085 (completed — relation query layer), DE-053 (TUI MVP — completed)
- **Constraints**: ArtifactSnapshot and ArtifactEntry are not modified. Search module is self-contained.
