## 1. Summary

- **Delta**: DE-117 — CLI boilerplate reduction
- **Design Revision**: DR-117
- **Specs Impacted**: SPEC-110 (supekku/cli)
- **Desired Outcome**: ~700 net lines removed from CLI layer; all repeated patterns extracted to shared helpers, decorator, enums, or factories.

## 2. Context & Constraints

- **Current Behaviour**: 8 boilerplate patterns repeated across CLI commands (format validation 13×, JSON flag 21×, error handling 17×, backlog list passthroughs 4×110 lines, backlog create 4×33 lines, find bypasses 6×, show lambdas 3×).
- **Target Behaviour**: Per DR-117 — `OutputFormat` enum, `resolve_format()`, `handle_cli_errors()` decorator, closure-based factories for backlog list/create, `find_artifacts()` consolidation, formatter extraction.
- **Dependencies**: DE-114 (completed). Parallel with DE-115.
- **Constraints**: CLI contract unchanged. `--help` output preserved. All existing tests pass.
