## 1. Summary

- **Delta**: [DE-127](./DE-127.md)

## 2. Context & Constraints

- **Current Behaviour**: `core/` contains 24 modules, 2 of which have 21
  combined cross-area imports, making the outer layer contract aspirational.
- **Target Behaviour**: `core/` is genuinely foundational; the two integration
  hubs live in orchestration.
- **Dependencies**: DE-125 (completed) — established migration pattern, contracts,
  and the re-export shim approach.
- **Constraints**:
  - No CLI behaviour changes.
  - Re-export shims preserve existing import paths.
  - Follow DE-125 migration pattern
    (see `mem.pattern.architecture.domain-migration`).
