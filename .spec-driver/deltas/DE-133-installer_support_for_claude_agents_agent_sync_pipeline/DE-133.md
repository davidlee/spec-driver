## 1. Summary & Context

- **Change Driver**: IMPR-031 — agent definitions (e.g. `dispatch-worker.md`) live in `.claude/agents/` but are not managed by the installer
- **Implementation Plan**: [IP-133](./IP-133.md)

## 2. Motivation

`spec-driver install` syncs skills, memories, hooks, and Claude config from the package source into the workspace. Agent definitions are the remaining gap — they're committed to git but manually placed. This creates drift risk when the package ships updated agent definitions.

## 3. Scope & Objectives

- **Primary Outcomes**:
  - Agent definitions in `supekku/agents/` are copied to `.claude/agents/` on install
  - Installer-owned semantics: managed agents overwritten on install, user-created agents left untouched
  - Dry-run support consistent with other install steps
- **Dependencies**: None — existing install infrastructure is sufficient

## 4. Out of Scope

- `.agents/` (Codex target) sync — future work
- Allowlist/manifest for agent filtering — not needed yet (only one managed agent)
- `.codex/agents/` or other agent host targets

## 5. Approach Overview

- **Package source**: `supekku/agents/*.md`
- **Install target**: `.claude/agents/` (direct copy, no symlink chain)
- **Ownership model**: files from package source are installer-owned and overwritten; user-created agents (not in package source) are left untouched
- **Key changes**:
  1. Create `supekku/agents/` and move `dispatch-worker.md` there as the package source
  2. Add `_install_agents()` to `install.py` following the `_install_claude_config` pattern
  3. Call from `initialize_workspace()`

## 6. Verification Strategy

- **VT**: Unit tests for `_install_agents()` — new, updated, unchanged, dry-run scenarios
- **VA**: Manual `spec-driver install --dry-run` confirms agent sync output

## 7. Risks & Mitigations

- **Risk**: Moving `dispatch-worker.md` out of `.claude/agents/` breaks existing installs – _Likelihood_: low – _Impact_: low – _Mitigation_: The installer will re-populate `.claude/agents/` from the new source on next install
