- **Dependencies**:
  - **Hard**: DE-118 (closed), DE-137 (closed — AUD-026 conformance evidence).
  - **Independent of**: per-artefact siblings DE-139..DE-142.
- **Constraints**:
  - POL-003 + DR-136 §11.1 — migration step package may not import `supekku.*` (import-linter `Migrations isolation` contract). Render emitters frozen-local per DEC-138-12.

  - Tolerant-on-read holds until per-kind strict-flip (DEC-138-04).
