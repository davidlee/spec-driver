## 1. Summary
- **Delta**: DE-035 - assembly-only tech specs and unit-spec deprecation
- **Specs Impacted**: ADR-003, PROD-012, PROD-015
- **Desired Outcome**: Simplify tech-spec workflows to assembly-first while preserving compatibility for legacy unit metadata.

## 2. Context & Constraints
- **Current Behaviour**: Dual taxonomy (unit + assembly) is implemented across sync, list, index, and validation workflows.
- **Target Behaviour**:
  - No new unit spec generation (`PROD-015.FR-002`, `PROD-012.FR-005`).
  - Assembly-first navigation/filtering (`PROD-015.FR-003`, `PROD-015.FR-004`).
  - Deprecation guidance for legacy unit metadata (`PROD-015.FR-005`).
- **Dependencies**:
  - RE-018 specification revisions.
  - ADR-003 follow-up decision (supersede or amend).
- **Constraints**:
  - Non-breaking rollout.
  - Warn-only migration posture for legacy repositories.
