## 1. Summary
- **Delta**: DE-044 - Centralize hardcoded workspace directory paths
- **Design Revision**: [DR-044](./DR-044.md)
- **Problems**: PROB-003 (magic path strings)
- **Policies**: POL-002 (avoid magic strings)
- **Desired Outcome**: All workspace directory names defined once in `core/paths.py`; zero magic path strings in production code.

- **Dependencies**: None
- **Constraints**: Pure refactor — zero behaviour change
