if you don't know where the current task card is:
   `uv run spec-driver find card` $ARGUMENTS

update the task card with implementation notes.

be concise, but record:
- what's done
- any surprises encountered or adaptations required
- any potential rough edges, omissions, or refactorings for later
- any follow-up actions advisable
- any open questions relating to completed or upcoming work
- the results of any /parity checks already performed
- relevant commit hash(es), or: the presence of uncommitted work
- if `just check` was run successfully since code was last modified (or any
  outstanding errors)
