# supekku.scripts.lib.blocks.verification_metadata

Metadata definition for verification coverage blocks.

This module defines the metadata schema for verification coverage blocks,
enabling metadata-driven validation and JSON Schema generation.

## Constants

- `PHASE_ID_PATTERN`
- `REQUIREMENT_ID_PATTERN`
- `SUBJECT_ID_PATTERN` - Regex patterns for validation
- `VERIFICATION_COVERAGE_METADATA` - Metadata definition for verification coverage blocks
- `VERIFICATION_ID_PATTERN` - Regex patterns for validation
- `__all__`
